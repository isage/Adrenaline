# Payloadex (Old)

The payloadex is a software that patches boot routines so unsigned custom modules can be loaded and executed. This one is made specifically to handle EPI boot modes and load the eCFW.

This is the older implementation. It is more minimal and smaller in size. Maintained in case issues comes up on newer implementation.
